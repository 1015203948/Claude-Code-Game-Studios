## Purpose & Player Need

暂停菜单让玩家在任意时刻停止游戏、检查或调整设置、保存进度。

**核心需求**：
- 随时暂停游戏时间
- 调整音量、震动、文字大小等偏好
- 保存进度并返回主菜单
- 快速恢复游戏

**非需求**：
- 暂停菜单不修改任何游戏数据（只读）
- 不提供游戏内帮助（由其他 UI 负责）

## Player Context on Arrival

### 从星图暂停
玩家正在查看星图、规划战略时需要暂停。暂停时星图保持显示（静态）。

### 从驾驶舱暂停
玩家在战斗中需要暂停。暂停时战斗冻结（敌人、飞船、时间全部暂停）。

### 返回后
Tap「继续游戏」，游戏立即从暂停点恢复（无加载，无过渡动画）。

## Entry & Exit Points

| 入口 | 触发 | 行为 |
|------|------|------|
| 星图 Tap [暂停] | 在星图视图 Tap 暂停按钮 | 暂停菜单覆盖星图 |
| 驾驶舱 Tap [暂停] | 在驾驶舱 HUD Tap 暂停按钮 | 暂停菜单覆盖驾驶舱 |
| 系统返回键 | Android 返回键 | 等同 Tap「继续游戏」 |
| 背面触控 | Android 背面（若有） | 预留 |

| 出口 | 目的地 | 行为 |
|------|--------|------|
| 继续游戏 | 星图/驾驶舱 | 关闭菜单，游戏继续 |
| 保存并退出 → 确认 → 执行 | 主菜单 | 保存进度，退出到主菜单 |
| 设置 → [返回] | 暂停主菜单 | 返回暂停菜单 |

```
暂停菜单
├── [继续游戏] — Tap → 关闭菜单，继续游戏
│
├── [设置]     — Tap → 进入设置子页面

├── 遮罩 Tap     — 等同 Tap「继续游戏」

│

    ├── [← 返回]     — Tap → 返回暂停主菜单

    ├── 震动开关     — Tap → 切换开关
```

## Transitions & Animations

| 动画 | 时长 | 缓动 | 说明 |
|------|------|------|------|
| 遮罩淡入 | 200ms | ease-out | 半透明黑色 |
| 面板缩放入场 | 300ms | scale 0.9→1.0 + fade | 从中心放大 |
| 面板缩放退出 | 200ms | scale 1.0→0.95 + fade | 缩小消失 |
| 按钮按下 | alpha 1.0→0.7 | 50ms linear | |
| 确认弹窗入场 | 200ms | scale 0.9→1.0 + fade | |
| 确认弹窗退场 | 150ms | fade out | |
| 滑块拖动 | 实时 | 无缓动 | |
| 开关切换 | 100ms | 状态翻转 | |

**禁止**：面板不使用滑入滑出动画（节省空间，减少等待感）。

## Accessibility

| 要求 | 实现 |
|------|------|
| A1: 热区 | 所有按钮/滑块热区 ≥48×48dp |
| A2: 色盲 | 「保存并退出」按钮有红色文字+警告图标双重标识 |
| A3: 文字缩放 | 所有文字支持系统字体缩放；设置中的文字大小选项会影响游戏内 UI |
| A4: 震动反馈 | 开关切换时震动反馈；关闭震动时仍显示视觉切换 |
| 对比度 | 面板背景与文字对比度 ≥4.5:1 |
| A5: 系统返回键 | Android 返回键功能明确（等同于继续游戏） |

**额外无障碍**：高对比度模式（设置中可开关）。

## Acceptance Criteria

- [ ] 从星图和驾驶舱均可通过暂停按钮打开暂停菜单
- [ ] 暂停菜单出现时 `Time.timeScale = 0`，游戏完全冻结
- [ ] Tap「继续游戏」关闭菜单，`Time.timeScale = 1`，游戏立即继续
- [ ] Tap「设置」进入设置子页面
- [ ] 设置子页面可调节：音乐音量、音效音量、震动开关、文字大小
- [ ] 所有设置实时保存到 PlayerPrefs
- [ ] Tap「保存并退出」弹出确认对话框
- [ ] 确认对话框点击外部/遮罩不可关闭，必须使用按钮
- [ ] 确认后保存进度并返回主菜单
- [ ] 所有按钮最小触控热区 48×48dp
- [ ] 按下时按钮 alpha 变为 0.7
- [ ] 面板使用 300ms 缩放入场动画
- [ ] 支持 Android 系统返回键（等同于继续游戏）
- [ ] 文字支持系统字体缩放
