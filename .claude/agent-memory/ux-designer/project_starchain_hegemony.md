安卓触屏策略游戏，支持 16:9（手机）和 4:3（平板）双屏比例。目标用户 18-35 岁硬核玩家，有 X4/Stellaris 背景。

**Why:** 平台和用户定义决定了所有 UX 决策的优先权重——触控优先、无 hover 状态、双比例适配是硬约束，不可为视觉美学妥协。

**How to apply:** 每次提出 UI 方案时，先验证：(1) 是否支持单手拇指触达；(2) 是否同时适配 16:9 和 4:3；(3) 是否无 hover 依赖。

## 游戏层级

- **星图层**：策略视角，1-2 次点击完成所有操作（游戏支柱 2）
- **驾驶舱层**：飞行/战斗视角，FPV 和 TPV 双视角可随时切换，虚拟摇杆操控

## Art Bible 已确认约束

- 面板禁止圆角矩形，使用硬切角/斜切角
- 触控目标最小 44×44dp
- 面板入场动画 0.4s 渐变淡入
- 主要文字 ≥7:1 对比度，次要文字 ≥4.5:1
- 亮度 ≥70% HSL 为主体，≤30% 为次要
- "亮度=优先级"代替文字标签的信息系统
